# Como rodar (passo a passo )

1. Cole o arquivo `.env` na raiz da pasta do projeto, junto com o `run.py`.

2. Abra o cmd na pasta do projeto e rode:

```cmd
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

3. Inicie o servidor dando duplo clique no arquivo `run.py` dentro da pasta do projeto (ou clique direito → Abrir com → Python).

4. Acesse no navegador: **http://localhost:5000**
